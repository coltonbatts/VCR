# ASCII Animations for Release

## Recommended: ascii.co.uk Imports

**For impressive, legible ASCII art** (like ballet, water droplet, 3D tunnel), use the ascii.co.uk import workflow:

```bash
./scripts/ascii_link_overlay.sh "https://www.ascii.co.uk/animated-art/<slug>-animated-ascii-art.html" -- --width 1920 --height 1080 --fps 24
```

See **[docs/ASCII_CO_UK_IMPORT.md](../docs/ASCII_CO_UK_IMPORT.md)** for the full guide and batch import examples.
