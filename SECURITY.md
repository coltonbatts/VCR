# Security Policy

## Supported Versions

Currently, only the latest release of VCR is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately. Do not open a public issue.

To report a vulnerability, please email **<coltonbatts@example.com>** (replace with your actual email).

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary. Please do not disclose the vulnerability publicly until a fix has been released and you've been notified.
